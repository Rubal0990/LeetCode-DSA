# Power of Two
Given an integer `n`, return <i>`true` if it is a power of two. Otherwise, return `false`</i>.

An integer `n` is a power of two, if there exists an integer `x` such that `n == 2x`.

## Examples
```
Input: n = 1
Output: true
Explanation: 2^0 = 1
```
```
Input: n = 16
Output: true
Explanation: 2^4 = 16
```
```
Input: n = 3
Output: false
```
```
Input: n = 4
Output: true
```
```
Input: n = 5
Output: false
```

## Constraints
* -2<sup>31</sup> <= n <= 2<sup>31</sup> - 1
