# Longest Palindromic Substring
Given a string `s`, return <i>the longest palindromic substring in </i>`s`.

## Examples
```
Input: s = "babad"
Output: "bab"
Note: "aba" is also a valid answer.
```
```
Input: s = "cbbd"
Output: "bb"
```
```
Input: s = "a"
Output: "a"
```
```
Input: s = "ac"
Output: "a"
```

## Constraints
* `1 <= s.length <= 1000`
* `s` consist of only digits and English letters.
